# Network Security Groups (NSGs) and Inspecting Network Protocols

## Overview

This project demonstrates how Azure Network Security Groups (NSGs) control network traffic and how network communication can be analyzed using Wireshark. Various protocols were captured and inspected while testing inbound and outbound security rules.

## Technologies Used

- Microsoft Azure
- Network Security Groups (NSGs)
- Wireshark
- Remote Desktop Protocol (RDP)

## Operating Systems Used

- Windows 10 Pro

## Skills Demonstrated

- Network Traffic Analysis
- Packet Inspection
- Azure Networking
- Network Security
- Protocol Analysis
- Troubleshooting Connectivity Issues

## Protocols Examined

- ICMP
- TCP
- UDP
- DNS
- RDP
- SSH

## Lab Objectives

- Deploy Azure Virtual Machines
- Install Wireshark
- Capture Network Traffic
- Analyze Protocol Behavior
- Configure NSG Rules
- Validate Security Controls
